 Mark a Task as Done:
  • Enter "/done 1" to mark the task you see in the display with a numbering of 1 as done.

Mark a Task as Done using Hotkey:
  • Press ctrl-s and the command "/done " will automatically appear in your command line box.
  • Simply type "1" and enter to mark the task you see in the display with a numbering of 1 as done.
