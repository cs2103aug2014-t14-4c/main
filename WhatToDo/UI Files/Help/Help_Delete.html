 Delete a Task:
  • Enter "/delete 1" to delete the task you see in the display with a numbering of 1.

Delete a Task using Hotkey:
  • Press ctrl-d and the command "/delete " will automatically appear in your command line box.
  • Simply type "1" and enter to delete the task you see in the display with a numbering of 1.
