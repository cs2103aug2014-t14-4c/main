Back              Help - Filter through Tasks!

 Filter by done:
  • Enter "/filter done" to see only done tasks.
  • Enter "/filter udone" to see only udone tasks. By default, WhatToDo will only show undone tasks at startup.
  • Enter "/filter nodone" to see both done and undone tasks.

Filter by type of task:
  • Enter "/filter fixed" to see only tasks with timings.
  • Enter "/filter due" to see only tasks with due dates.
  • Enter "/filter notype" to see all your tasks.

Filter by date:
  • Enter "/filter today" to see only tasks beginning from today. Similarly, enter "/filter 30 Nov" to see only tasks beginning from 30 Nov.
  • You can also enter "/filter today to tomorrow" to see only tasks between today and tomorrow.
  • You can also enter "/filter nodate" to see only your floating tasks.

Filter with help of hotkey:
  • Press ctrl-q and the command "/filter " will automatically appear in your command line box.
  • You can then proceed to type your desired filter parameters.
