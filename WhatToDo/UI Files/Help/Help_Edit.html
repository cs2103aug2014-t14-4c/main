 Edit a Task:
  • Enter "/edit 1 Finish tutorials" to change the task you see in the display with a numbering of 1 to the task called "Finish tutorials".

Edit a Task using Hotkey:
  • Press ctrl-e and the command "/edit " will automatically appear in your command line box.
  • Simply type "1 Finish tutorials" and enter to change the task you see in the display with a numbering of 1 to the task called "Finish tutorials".
